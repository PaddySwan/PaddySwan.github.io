Accessibility Considerations

Ensure all icons have alt tags. Colors should match those found in the foundational colors so that they meet the color-contrast requirements.

Usage

  • Primary usage is for a skills list for a portfolio. However, this may be used for listing various things.

  • This is not a section on its own, this should be within a section on a page.
