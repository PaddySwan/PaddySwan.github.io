Typography

The typographical hierarchy for this design system combines Merriweather and Merriweather Sans for both the headers and body text. The base font size is 16px in a major-third scale. Merriweather is a great typeface to use because it balances sharp and soft edges to make the letters distinct. These two pair well together to make a complete typographical hierarchy. Color is not defined for these headers other than the hyperlink text. Color should be chosen based on the page's color scheme and dark/light mode decisions.
